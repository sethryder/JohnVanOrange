    Introduction to John VanOrange

    This code can be used to create a site to view random images. Follow the instructions below to setup this repo on your own server. PHP 5.2 and MySQL are required, and Apache is the assumed web server.

    To setup this repo, do the following:

    $ git clone https://github.com/cbulock/JohnVanOrange.git
$ cd JohnVanOrange
$ mkdir /smarty/templates_c/
$ chmod +w /smarty/templates_c/
$ chmod +w /media/
$ chmod +w /media/thumbs/
$ chmod +w /upload-handler/files/
$ chmod +w /upload-handler/thumbnails/
$ mysql -uroot -e "create database 'jvo'"
$ mysql jvo < /sql/tables.sql
$ cp settings.inc.default settings.inc

    Finally:

    • Database user needs to be setup.
    • The settings.inc needs to have the the correct database and various settings input
    • PHP needs the Imagick extension be setup.

    This is the codebase for the image viewing site at http://johnvanorange.com
